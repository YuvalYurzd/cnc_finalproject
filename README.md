## Requirements
* Linux (Ubuntu 22.04 LTS recommanded)/Windows machine
* Python 3
* pyodbc library
* Root/Admin privileges

## Running (Windows)
```
# DHCP client & server
python DHCP.py
python client.py

# DNS client & server
python dnsServer.py
python client.py

# SQL client & server (TCP)
python sqlServer.py
python client.py

# SQL client & server (Reliable UDP)
python sqlServer.py
python rudp_client.py

```

## Running (Linux)
```
# DHCP client & server
python3 DHCP.py
python3 client.py

# DNS client & server
python3 dnsServer.py
python3 client.py

# SQL client & server (TCP)
python3 sqlServer.py
python3 client.py

# SQL client & server (Reliable UDP)
python3 sqlServer.py
python3 rudp_client.py

```
